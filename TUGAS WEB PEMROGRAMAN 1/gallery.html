HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan mengatur konten pada halaman web. HTML digunakan untuk membuat struktur dasar halaman web dan mendefinisikan elemen-elemen pada halaman web, seperti teks, gambar, audio, video, tabel, formulir, dan lain-lain.
HTML terdiri dari serangkaian tag atau elemen yang ditempatkan di dalam dokumen HTML. Setiap tag dimulai dengan karakter "<" dan diakhiri dengan karakter ">". Isi atau konten tag ditempatkan di antara tag pembuka dan tag penutup. Contohnya, tag pembuka untuk menampilkan judul pada halaman web adalah dan tag penutupnya adalah . Isi judul ditempatkan di antara tag pembuka dan tag penutup seperti ini: "Ini adalah judul halaman". Dengan menggunakan HTML, Anda dapat membuat struktur dan tata letak halaman web, menambahkan teks dan media, membuat tautan dan formulir, serta mengatur warna dan gaya halaman web. HTML juga dapat digunakan bersamaan dengan CSS (Cascading Style Sheets) dan JavaScript untuk meningkatkan tampilan dan fungsionalitas halaman web.

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web yang dibuat dengan HTML dan XHTML. CSS memungkinkan pengguna untuk memisahkan tampilan dari isi dokumen, sehingga pengguna dapat membuat tampilan halaman web yang konsisten dan mudah dikelola. CSS berfungsi sebagai pemisah antara konten HTML dan tampilan visual halaman web, sehingga pengguna dapat memperbarui tampilan halaman web tanpa harus memodifikasi konten HTML. CSS berisi serangkaian aturan atau deklarasi gaya yang diterapkan pada elemen HTML di halaman web. Deklarasi gaya CSS terdiri dari dua bagian: selector dan property. Selector menentukan elemen HTML mana yang akan diterapkan gaya CSS, sementara property menentukan bagaimana elemen tersebut akan ditampilkan. Contohnya, selector "h1" dapat diterapkan untuk mengatur gaya pada elemen HTML "h1" pada halaman web, dan property "color" dapat digunakan untuk menentukan warna teks pada elemen "h1". Dengan menggunakan CSS, pengguna dapat mengubah warna, ukuran, posisi, tata letak, dan banyak elemen tampilan lainnya pada halaman web. CSS juga mendukung penggunaan media query, yang memungkinkan pengguna untuk menyesuaikan tampilan halaman web sesuai dengan ukuran layar atau jenis perangkat yang digunakan. CSS merupakan salah satu komponen penting dalam desain web modern, dan setiap pengembang web harus memahami dasar-dasar CSS.

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang dirancang khusus untuk pengembangan web dinamis. PHP digunakan untuk membuat aplikasi web yang dapat menghasilkan konten dinamis berdasarkan permintaan pengguna. Bahasa ini sangat populer di antara pengembang web karena mudah dipelajari, gratis, dan kompatibel dengan sebagian besar server web dan database. PHP berfungsi sebagai bagian dari server web dan digunakan untuk menghasilkan halaman web dinamis dengan cara mengambil informasi dari database atau file, dan mengirimkannya ke browser sebagai HTML. PHP juga mendukung pemrograman berorientasi objek (OOP) dan dapat digunakan untuk membuat fungsi, kelas, dan objek yang dapat digunakan kembali pada berbagai proyek web. Beberapa fitur PHP yang populer termasuk kemampuan untuk memproses formulir web, memvalidasi input pengguna, membuat dan mengelola sesi pengguna, menghasilkan dan mengirim email, dan mengakses dan memanipulasi data dalam database. PHP juga memungkinkan integrasi dengan teknologi lain seperti JavaScript dan XML. PHP dapat dijalankan pada sebagian besar sistem operasi termasuk Windows, macOS, Linux, dan platform UNIX lainnya. Bahasa ini juga didukung oleh sebagian besar server web seperti Apache, Nginx, dan Microsoft IIS. PHP telah digunakan oleh banyak perusahaan dan situs web terkenal seperti Facebook, Wikipedia, dan WordPress.

MySQL adalah sistem manajemen basis data relasional (RDBMS) open source yang sangat populer. RDBMS adalah jenis sistem manajemen basis data yang didasarkan pada model data relasional, di mana data disimpan dalam bentuk tabel yang terkait satu sama lain melalui kunci. MySQL adalah salah satu RDBMS tercepat, paling andal, dan paling fleksibel di dunia. MySQL memiliki kemampuan untuk menangani volume data yang besar, serta mendukung banyak pengguna dan aplikasi yang berbeda. MySQL mendukung bahasa SQL (Structured Query Language) yang populer, dan juga mendukung beberapa ekstensi SQL yang lebih canggih seperti pemrosesan transaksi, penjadwalan, dan pengelolaan transaksi. MySQL juga dapat berintegrasi dengan banyak bahasa pemrograman seperti PHP, Python, Ruby, dan Java, yang memungkinkan pengembang untuk membuat aplikasi web yang dapat mengambil, menyimpan, dan memproses data dari MySQL. MySQL juga dapat digunakan dengan banyak platform sistem operasi, termasuk Windows, macOS, dan Linux. MySQL telah digunakan oleh banyak organisasi terkenal di seluruh dunia, termasuk Google, Facebook, Twitter, dan YouTube. MySQL adalah pilihan populer bagi pengembang web karena mudah digunakan, cepat, andal, dan open source.
